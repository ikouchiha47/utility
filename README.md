# utility
utility scripts for work and other stuff

### installation
- `git clone https://github.com/ikouchiha47/utility.git`
- `cd utility`
- `bash install.sh`

### manual installation
- chmod +x filename
- add the file to $PATH (optional)

### minimum requirements
python 2.7.13 or more
